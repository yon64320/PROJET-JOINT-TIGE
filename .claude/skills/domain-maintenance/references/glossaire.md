# Glossaire métier — Maintenance industrielle

| Terme | Définition |
|-------|------------|
| **Arrêt d'unité** | Arrêt planifié d'une unité de production pour maintenance. Aussi : turnaround, shutdown, TA. |
| **OT** | Ordre de Travail — une tâche de maintenance sur un équipement |
| **LUT** | Liste Unifiée de Travaux — fichier mère listant tous les OTs d'un arrêt |
| **J&T** | Joint & Tige — suivi de toutes les brides touchées et leur boulonnerie |
| **Gamme** | Séquence d'opérations phase par phase pour exécuter un OT |
| **ITEM** | Numéro d'identification d'un équipement (ex: B-E01, S-C02) |
| **Bride** | Assemblage boulonné reliant deux tuyaux/équipements. Contient un joint d'étanchéité. |
| **Joint** | Pièce d'étanchéité entre deux brides. Types : provisoire (pendant travaux) et définitif (remontage final). |
| **Platine** | Joint plein (JP) — plaque métallique pour isoler un circuit |
| **Tige** | Boulon traversant (tige filetée + écrous) maintenant les brides ensemble |
| **DN** | Diamètre Nominal — taille de la bride/tuyau en mm |
| **PN** | Pression Nominale — classe de pression en bar |
| **CL** | Class (ANSI) — CL150 ≈ PN20, CL300 ≈ PN50, CL600 ≈ PN100 |
| **RF** | Raised Face — type de face de bride (surélevée) |
| **RTJ** | Ring Type Joint — face de bride à anneau métallique |
| **BP** | Bride Pleine — bride sans trou central (pour isoler) |
| **JP** | Joint Plein — platine (synonyme) |
| **TB** | Travaux de Base — inclus dans le forfait |
| **TC** | Travaux Complémentaires — hors forfait |
| **TA** | Travaux Annulés |
| **ESP** | Équipement Sous Pression — soumis à réglementation |
| **ATEX** | ATmosphère EXplosive — zone à risque d'explosion |
| **Ubleam** | Système de tag NFC/QR pour identifier les brides sur le terrain |
| **ADF** | À Définir / Approvisionnement à Définir |
| **Donneur d'ordres** | Le client (raffinerie, pétrochimie) qui commande les travaux |
| **BUTA** | Abréviation de Butachimie (nom de l'unité / du client) |
| **EMIS** | Entretien Maintenance Industrielle et Service — l'entreprise |
